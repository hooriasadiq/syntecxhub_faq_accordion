# 🚀 Modern FAQ Accordion Widget

A clean and modern FAQ Accordion built using **HTML, CSS, and JavaScript** with smooth animations and responsive design.

---

## ✨ Features

* ✅ Expand / Collapse FAQ items
* 🎨 Modern Glassmorphism UI
* ⚡ Smooth CSS animations
* 📱 Fully Responsive (Mobile Friendly)
* 🔄 Single open item (better UX)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── style.css
```

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in browser
